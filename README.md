# Link in Bio Project

## 🚀 Upload ke GitHub
1. Buat repo baru
2. Upload semua file
3. Commit & push

## ☁️ Deploy ke Cloudflare Pages
1. Login Cloudflare
2. Pages → Create Project
3. Connect GitHub repo
4. Build: none
5. Output: /

## ✏️ Edit Link
Edit file `/data/links.json`

## 🌐 Custom Domain
Settings → Domain → Add domain

## 📊 Analytics (Umami)
Tambahkan script Umami di index.html
